## 벤포드의 법칙이란?
간단히 말하면, 숫자의 첫 자리가 **1**일 확률이 가장 크고, **9**일 확률은 가장 적다는 법칙입니다. 예를 들어, 숫자가 **1**로 시작하는 경우가 훨씬 많고, **9**는 거의 나오지 않고 첫 자리가 1일 확률이 30%에 가깝습니다.
